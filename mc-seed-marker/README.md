# MC Seed Marker

Minecraft Seed 地圖標記工具，可以為你的世界添加自訂標記。

## 功能

- 輸入 Seed 後顯示座標網格
- 雙擊地圖快速新增標記
- 手動輸入 X/Z 座標新增標記
- 8 種顏色可選
- 標記自動存到 localStorage
- 滾輪縮放、拖曳移動
- 點擊標記列表跳轉到該位置
- 直接連結到 ChunkBase 查看結構

## 使用方式

### 本機測試

```bash
# 安裝依賴
npm install

# 開發模式
npm run dev
```

打開 http://localhost:3000

### 部署到 Vercel

1. Push 到 GitHub
2. 連接 Vercel
3. 設定自訂網域 mc.cayoh.run

## 操作說明

- **載入 Seed**: 輸入數字後按「載入」
- **新增標記**: 雙擊地圖，或填寫右側表單
- **單擊地圖**: 自動填入 X/Z 座標到表單
- **拖曳**: 按住左鍵拖曳移動視角
- **縮放**: 滾輪或點擊 +/- 按鈕
- **回原點**: 點擊 ⌂ 按鈕
- **跳轉標記**: 點擊標記列表中的項目
- **刪除標記**: 點擊 ✕ 按鈕
